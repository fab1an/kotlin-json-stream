JsonWriter

class JsonWriter(sink: BufferedSink, val prettyPrint: Boolean = false)

Writes a JSON (RFC 7159) encoded value to a stream. This stream includes both literal values (strings, numbers, booleans, and nulls) as well as the begin and end delimiters of objects and arrays.
